Let's get started

How to build and use our API ?

First Step - Create an IFTTT account

First, you have to create an account on the IFTTT website. Register or create your account and click on "New Applet". Then, click on "+this" to select your trigger and search & select "Webhooks". Click on the "connect" button, then on"receive a web request".

Second Step - Name the Event

Name your event "Notify" and click on "Create Trigger". If you send the trigger, you need to sort of specify what happens after that.Click on "+that" to select the action to accomplish.

Third Step - Set up the "THAT" part

IFTTT offers a lot of services: SMS, emails, phone calls, even publishing posts on Wordpress, Tumblr and so on. What we are looking for is "Notifications". What it does is that if you have a mobile phone and that you have the IFTTT app installed in it, you can set a notification that will pop up on your phone.

Step Four - Choose action

Select "Send Notification from the IFTTT App" then go to the next step

Step Five - Complete Action Fields

You can replace the text; the notification that will pop up will show the text that is described under "Message". You can replace it by whatever you want. You can also pass values to these notifications by clicking on "Adding an Ingredient". You can add {{Value 1}}, {{Value 2}} as arguments. The notification will pop up the event name has occurred with the values 1 and 2. Values could be location, an IP Adress, it can be anything.
Click on "Create Action" and then "Finish"

Final Step - Trigger the Event

Good job ! Now the applet is created. IFTTT Webhooks work with HTTP requests. In order to associate these requests with your account and your applets, IFTTT gives a unique key by account. Ypu need to have a look at the Webhooks documentation to access your key. hello-ifttt is the Event Name we set in Step 2. value1, value2, value3 can take any value and will replace Value1, Value2 and Value3 from Step 5.
You can send a POST or a GET statement, but to use the command line, you need to try it with "curl". Just open your terminal and past the code.Hit enter, and if everithing went fine, you should see "Congratulations ! You've fired the Notify event" and that should trigger a notification on your phone.
